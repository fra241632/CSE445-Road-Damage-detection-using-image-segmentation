# Road Damage Detection and Lane Segmentation

## Overview

This project aims to detect:

- Road Cracks
- Potholes
- Open Manholes

using YOLO object detection, and perform lane segmentation using a deep learning segmentation model.

The project is being developed in **Google Colab** using datasets from **Kaggle**.

---

## Technologies

- Python
- Google Colab
- PyTorch
- Ultralytics YOLO
- OpenCV
- Google Drive
- Kaggle

---
